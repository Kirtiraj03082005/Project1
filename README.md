# Project1
Pong Game

This project is a classic implementation of the Pong game, developed using C++ with a focus on clean structure, modular design, and real-time gameplay mechanics. It recreates one of the earliest and most iconic arcade games, where players control paddles to hit a moving ball and score points against an opponent.

The goal of this project is not only to build a playable game but also to demonstrate fundamental concepts of game development such as collision detection, object-oriented design, rendering, and user input handling.

The codebase is organized into separate components (such as ball and bat/paddle logic), making it easy to understand, extend, and maintain. Whether you're a beginner exploring game development or someone interested in how classic games are built from scratch, this project serves as a simple yet effective learning resource.
